# Spotify Playlist Visualizer 🎧

A tool to explore and visualize Spotify playlist metadata using Streamlit.

> ❗ This tool **does not download** any music. It uses the Spotify API to fetch and display playlist data for educational and visualization purposes only.

---

## 📌 Features

- 🔍 View playlist info: tracks, artists, duration
- 📈 Visualize track popularity and audio features
- 🖥️ Web version (Streamlit) + GUI version (Tkinter)
- 📁 Export playlist metadata (CSV/JSON)

---

## 💻 Tech Stack

- Python
- Streamlit
- Spotipy (Spotify Web API)
- Tkinter (GUI)
